What is the definition of server port?
Web server ports are the logical endpoints of a network connection that is used to exchange information between a 
web server and a web client

What is server port range?
Port numbers range from 0 to 65536, but only ports numbers 0 to 1024 are reserved for privileged services and 
designated as well-known ports.
